Chapter 10. How to use content scripts.
You can use content scripts to run scripts on the page specified by the URL.
The target URL and script are defined in manifest.json.
This time, add a button with a script to the page with the following URL.
https://www.wikipedia.org
https://nory-soft.web.app

1. Prepare the necessary files.

The contents of the files are described on this Chapter.

You can also download from the following Chrome Extension.
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom?hl=en

2. Place the files to any folder on your PC.
Ex. C:\\ChromeExtenson\\Chapter10

3. Enter "chrome://extensions" in the chrome address bar. Enable Developer mode.

4. Click the "Load unpacked" to specify the folder.

5. Go to https://www.wikipedia.org in Chrome. Make sure you see the "Change background to red" button.


6. Click the "Change background to red" button to change the background to red.


Explanation of program.
file namedescription
manifest.jsonManifest file.
matches.jsA java script that runs when the URLs match.

manifest.json
{
  "name": "Chapter 10",
  "version": "1.0",
  "manifest_version": 2,

  "description": "This is Chapter 10",

  "content_scripts": [
    {
      "matches": ["*://*.wikipedia.org/*"],
      "js": ["matches.js"]
    },
    {
      "matches": ["*://nory-soft.web.app/*"],
      "js": ["matches.js"]
    }
  ]
}

matches.js
let button = document.createElement('input');
button.type = 'button';
button.value = 'Change background to red';

let child = document.body.firstChild;
document.body.insertBefore(button, child);

button.addEventListener("click", button_click);

function button_click()
{
  document.body.style.backgroundColor = 'red';
}
