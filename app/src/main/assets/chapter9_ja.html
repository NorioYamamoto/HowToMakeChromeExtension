9章 Alarmを使おう
Alarmを使うと指定した日時に処理を動かすことが出来ます。
今回は指定した秒の経過時にアラートを表示します。

1. 必要なファイルを準備します。

ファイルの内容はこの章に記載しています。
アイコンはペイント等のアプリケーションで作成して下さい。

以下のChrome拡張からファイルをダウンロードすることも出来ます。
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom

2. PCの任意のフォルダにファイルを置きます。
例 C:¥ChromeExtenson¥Chapter9

3. Chromeのアドレスバーに"chrome://extensions"を入力し、デベロッパーモードを有効にします。

4. "パッケージされていない拡張機能を読み込む"をクリックし、フォルダを指定します。

5. アドレスバーの右に追加されたアイコンをクリックします。
テキストボックスに秒を入力し、SetボタンをクリックするとAlarmがセットされます。
ClearボタンをクリックするとAlarmがクリアされます。


6. 指定した秒の経過時にアラートが表示されます。

myscript.js
const alarm_id = 'alarm_001';

document.getElementById('id_Set').onclick = function(){
  let dt = new Date();
  let sec = document.getElementById('id_Sec').value;
  dt.setSeconds(dt.getSeconds() + parseInt(sec));
  chrome.alarms.create(alarm_id, {when: dt.getTime()});
};

document.getElementById('id_Clear').onclick = function(){
  chrome.alarms.clear(alarm_id);
};
