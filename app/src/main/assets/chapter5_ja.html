5章 executeScriptを使おう
executeScriptは既存のウェブページでスクリプトを実行できます。
これにより既存のページを読み書きすることができます。

1. 必要なファイルを準備します。

ファイルの内容はこの章に記載しています。
アイコンはペイント等のアプリケーションで作成して下さい。

以下のChrome拡張からファイルをダウンロードすることも出来ます。
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom

2. PCの任意のフォルダにファイルを置きます。
例 C:¥ChromeExtenson¥Chapter5

3. Chromeのアドレスバーに"chrome://extensions"を入力し、デベロッパーモードを有効にします。

4. "パッケージされていない拡張機能を読み込む"をクリックし、フォルダを指定します。

5. アドレスバーの右に追加されたアイコンをクリックします。

6. Redボタンをクリックします。


7. 背景が赤になります。

popup.html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
</head>

<body style="min-width:100px">
<input type="button" id="id_Red" value="Red">
<br><br>
<input type="button" id="id_Blue" value="Blue">

<script src="myscript.js"></script>
</body>
</html>

myscript.js
// Click Red button
document.getElementById('id_Red').onclick = function() {
    chrome.tabs.executeScript(null, {
        "code": "document.body.style.backgroundColor='red'"
    })
//  // If you prefer external files.
//  chrome.tabs.executeScript(null, {
//      "file": "bg_red.js"
//  })
}
// Click Blue button
document.getElementById('id_Blue').onclick = function() {
    chrome.tabs.executeScript(null, {
        "code": "document.body.style.backgroundColor='Blue'"
    })
}
