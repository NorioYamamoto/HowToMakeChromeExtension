1章 初めてのChrome拡張
簡単なChrome拡張を作りましょう。
1. 必要なファイルを準備します。

ファイルの内容はこの章に記載しています。
アイコンはペイント等のアプリケーションで作成して下さい。

以下のChrome拡張からファイルをダウンロードすることも出来ます。
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom

2. PCの任意のフォルダにファイルを置きます。
例 C:¥ChromeExtenson¥Chapter1

3. Chromeのアドレスバーに"chrome://extensions"を入力し、デベロッパーモードを有効にします。

4. "パッケージされていない拡張機能を読み込む"をクリックし、フォルダを指定します。


5. アドレスバーの右に追加されたアイコンをクリックします。


6. "Hello world!"が表示されれば成功です。



プログラムの説明
ファイル名 説明
manifest.jsonマニュフェスト・ファイル
popup.htmlアイコンをクリックで表示
Chapter_1_19.png拡張機能アイコン Size:19x19

manifest.json
{
    "name": "Chapter 1",
    "short_name": "C 1",
    "version": "1.0",
    "manifest_version": 2,

    "description": "Chapter 1",

    "browser_action": {
        "default_icon": "Chapter_1_19.png",
        "default_title": "Chapter 1",
        "default_popup": "popup.html"
    }
}

popup.html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
</head>
<body style="min-width:250px">
Hello World!
</body>
</html>
