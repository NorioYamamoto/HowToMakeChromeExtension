Chapitre 9. Comment utiliser Alarm.
Si vous utilisez Alarm, vous pouvez lancer le processus à la date et à l'heure spécifiées.
Cette fois, une alerte sera affichée lorsque les secondes spécifiées se seront écoulées.

1. Préparez les fichiers nécessaires.

Le contenu des fichiers est décrit dans ce chapitre.
Créez le fichier icône avec une application telle que Paint.

Vous pouvez également télécharger à partir de l'extension Chrome suivante.
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom?hl=fr

2. Placez les fichiers dans n'importe quel dossier de votre PC.
Ex. C:\\ChromeExtenson\\Chapter9

3. Saisissez "chrome://extensions" dans la barre d'adresse Chrome. Activez le mode développeur.

4. Cliquez sur "Load unpacked" pour spécifier le dossier.

5. Cliquez sur l'icône ajoutée à droite de la barre d'adresse.
Entrez les secondes dans la zone de texte et cliquez sur le bouton Set pour définir l'alarme.
Cliquez sur le bouton Clear pour effacer l'alarme.


6. Une alerte s'affiche après les secondes spécifiées.

myscript.js
const alarm_id = 'alarm_001';

document.getElementById('id_Set').onclick = function(){
  let dt = new Date();
  let sec = document.getElementById('id_Sec').value;
  dt.setSeconds(dt.getSeconds() + parseInt(sec));
  chrome.alarms.create(alarm_id, {when: dt.getTime()});
};

document.getElementById('id_Clear').onclick = function(){
  chrome.alarms.clear(alarm_id);
};
