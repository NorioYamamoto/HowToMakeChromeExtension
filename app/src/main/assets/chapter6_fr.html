Chapitre 6. Comment utiliser les menus contextuels.
Enregistrez l'extension Chrome dans le menu contextuel de Chrome.

1. Préparez les fichiers nécessaires.

Le contenu des fichiers est décrit dans ce chapitre.
Créez le fichier icône avec une application telle que Paint.

Vous pouvez également télécharger à partir de l'extension Chrome suivante.
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom?hl=fr

2. Placez les fichiers dans n'importe quel dossier de votre PC.
Ex. C:\\ChromeExtenson\\Chapter6

3. Saisissez "chrome://extensions" dans la barre d'adresse Chrome. Activez le mode développeur.

4. Cliquez sur "Load unpacked" pour spécifier le dossier.

5. Sélectionnez un mot-clé sur n'importe quelle page et sélectionnez "Search Twitter for XXXXX" dans le menu contextuel.


6. Les résultats de la recherche sont affichés sur Twitter.

manifest.json
{
    "name": "Chapter 6",
    "short_name": "C 6",
    "version": "1.0",
    "manifest_version": 2,

    "description": "Chapter 6",

    "background": {
        "scripts": ["event.js"],
        "persistent": true
    },

    "permissions": [
        "contextMenus"
    ],

    "icons": {
        "19": "Chapter_6_19.png"
    }
}

event.js
chrome.contextMenus.create({
    "title": "Search Twitter for \"%s\"",
    "type": "normal",
    "contexts": ["selection"],
    "onclick": function(info) {
        chrome.tabs.create({
            url: 'http://twitter.com/search?q='+encodeURIComponent(info.selectionText)
        });
    }
});
