6章 contextMenusを使おう
Chromeのコンテキストメニュー（右クリックメニュー）にChrome拡張を登録することが出来ます。

1. 必要なファイルを準備します。

ファイルの内容はこの章に記載しています。
アイコンはペイント等のアプリケーションで作成して下さい。

以下のChrome拡張からファイルをダウンロードすることも出来ます。
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom

2. PCの任意のフォルダにファイルを置きます。
例 C:¥ChromeExtenson¥Chapter6

3. Chromeのアドレスバーに"chrome://extensions"を入力し、デベロッパーモードを有効にします。

4. "パッケージされていない拡張機能を読み込む"をクリックし、フォルダを指定します。

5. 任意のページでキーワードを選択し、コンテキストメニューから"Search Twitter for XXXXX"を選択します。


6. 検索結果がツイッターに表示されます。


プログラムの説明
ファイル名 説明
manifest.jsonマニュフェスト・ファイル
event.jsバックグラウンドで動くjavaスクリプト
Chapter_6_19.png拡張機能アイコン Size:19x19

manifest.json
{
    "name": "Chapter 6",
    "short_name": "C 6",
    "version": "1.0",
    "manifest_version": 2,

    "description": "Chapter 6",

    "background": {
        "scripts": ["event.js"],
        "persistent": true
    },

    "permissions": [
        "contextMenus"
    ],

    "icons": {
        "19": "Chapter_6_19.png"
    }
}

event.js
chrome.contextMenus.create({
    "title": "Search Twitter for \"%s\"",
    "type": "normal",
    "contexts": ["selection"],
    "onclick": function(info) {
        chrome.tabs.create({
            url: 'http://twitter.com/search?q='+encodeURIComponent(info.selectionText)
        });
    }
});
