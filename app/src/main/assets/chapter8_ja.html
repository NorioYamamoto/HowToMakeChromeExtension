8章 sendMessageを使おう
ドメイン間でデータ通信を行う場合はeventPage経由でsendMessageを使います。

ドメインA ---> eventPage ---> ドメインB

なお、localStorageのスコープはドメイン単位なのでドメイン間のデータ通信には使用出来ません。

1. 必要なファイルを準備します。

ファイルの内容はこの章に記載しています。
アイコンはペイント等のアプリケーションで作成して下さい。

以下のChrome拡張からファイルをダウンロードすることも出来ます。
https://chrome.google.com/webstore/detail/how-to-make-chrome-extens/kcekgmndcdbeflkloloihblbepjnpfom

2. PCの任意のフォルダにファイルを置きます。
例 C:¥ChromeExtenson¥Chapter8

3. Chromeのアドレスバーに"chrome://extensions"を入力し、デベロッパーモードを有効にします。

4. "パッケージされていない拡張機能を読み込む"をクリックし、フォルダを指定します。

5. アドレスバーの右に追加されたアイコンをクリックします。
ドメインAとドメインBが表示されます。これらはテスト用で実際には拡張機能内のローカルコンテンツです。

6. ドメインAを選択し、Sendボタンをクリックします。


7. ドメインBを選択し、Recvボタンをクリックします。ドメインAのデーターが受信されたことを確認して下さい。

myscript.js

  chrome.tabs.create({url:'domainB.html'}, tab => {});

domainA.js
document.getElementById('id_Send').onclick = function() {
  var value = document.getElementById('ta1').value;
  chrome.runtime.sendMessage({method: 'Send', key: 'key', value: value}, function (response) {
    ;
  });
};

domainB.js
document.getElementById('id_Recv').onclick = function() {
  chrome.runtime.sendMessage({method: 'Recv', key: 'key'}, function (response) {
    document.getElementById('tb1').value = response.value;
  });
};

event.js
var value;
chrome.runtime.onMessage.addListener(function (request, sender, sendResponse) {
  switch (request.method) {
  case 'Send':
    value = request.value;
    break;
  case 'Recv':
    sendResponse({value: value});
    break;
  }
});
